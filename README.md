# 環境（Environment）
- Build: Gradle（Groovy DSL）
- Spring Boot: 3.5.8
- Java: 21
- IDE: IntelliJ IDEA
- DB: PostgreSQL
- Mapper: MyBatis
- View: Thymeleaf
- Dependencies: Spring Web / Thymeleaf

# 目的（Purpose）
- Spring Boot + MyBatis + PostgreSQL を用いた CRUD アプリの基本構成を学習・整理する
- レイヤードアーキテクチャ（Controller / Service / Repository） の設計と実装を練習する
- 例外処理・入力バリデーション を実装し、保守性の高い構成を目指す

# 主要機能（Planned / In Progress）
- ユーザー登録・ログイン（予定）
- 記録（ログ）CRUD（作成 / 参照 / 更新 / 削除）
- 一覧表示（予定）
